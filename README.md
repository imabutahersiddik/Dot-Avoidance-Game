## 🎮 Quick Start Guide

1. **Choose Your Path:**
   ```
   [EASY] - For beginners (1x Score)
   [HARD] - For skilled players (2x Score)
   [VERY HARD] - For the fearless! (3x Score)
   ```

2. **Controls:**
   ```
   ⌨️ Keyboard Warriors:
   ↑ or W = Move Up
   ↓ or S = Move Down
   ← or A = Move Left
   → or D = Move Right
   
   📱 Touch Masters:
   Use on-screen D-pad
   ```

3. **Pro Tips:**
   - Stay away from red dots!
   - Watch their patterns
   - Time your movements
   - Don't get cornered!
